# Bio-Entity Extraction & Knowledge Base Construction for Disease Models
This project demonstrates a systematic approach to transforming unstructured biomedical literature into a structured knowledge base, specifically focusing on comparative medicine and mouse disease models.

1. Overview
In biomedical research, the bottleneck is often not the lack of data, but the lack of structured data. This repository contains a pipeline to extract, clean, and query key biological entities from recent molecular biology papers to facilitate strategic decision-making in R&D pipelines

2. Core Methodology
The project focuses on the 'Database Generation' process, ensuring that raw scientific text is converted into a machine-readable format suitable for R&D strategy tools.


Disease: Specific pathological condition or resistance state (e.g., TMZ-resistance)
Model:	Genetic background and experimental induction method
Key Target:	Primary molecular markers or signaling pathways (e.g., FLG, MGMT)
Species:	Taxonomic classification for comparative medicine analysis (e.g., C57BL/6J)
